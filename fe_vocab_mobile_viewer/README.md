# FE用語ビューア

CSVファイルをスマホで読み込み、用語カードをスワイプで切り替えて閲覧するHTMLビューアです。

## 使い方

1. `index.html`, `style.css`, `app.js` を同じフォルダに置く
2. スマホで `index.html` を開く
3. `CSV選択` を押して、自分のCSVファイルを選ぶ
4. カード画面で左右スワイプ、または下部の「前へ」「次へ」で移動

## 対応しているCSV列名

次の列名に対応しています。

- 用語
- 分野
- 一言説明
- 詳細説明
- 関連語
- 重要度

関連語は `;`、`；`、`,`、`、` のいずれかで区切るとタグ表示されます。

## iPhoneでの注意

ローカルHTMLからCSVを毎回読み込む方式です。
完全にアプリ化したい場合は、GitHub Pagesなどに置くか、PWA化するとホーム画面から開きやすくなります。
