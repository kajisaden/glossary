# FE用語ビューア CSV埋め込み版

CSV選択なしで、最初から用語データを読み込む版です。

## ファイル構成

- `index.html`
- `style.css`
- `app.js`
- `data.js`

## 読み込み済みデータ

- 元CSV: `fe_terms_completed_all.csv`
- 行数: 475

## 使い方

このフォルダをGitHubリポジトリに置き、GitHub Pagesで公開してください。
公開後は `index.html` を開くだけで用語データが表示されます。

## 更新方法

用語CSVを更新した場合は、`data.js` を作り直す必要があります。
