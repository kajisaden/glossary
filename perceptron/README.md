# パーセプトロンを体験して学ぼう

初心者がパーセプトロン（ニューラルネットワークの最も基本的な部品）を、手を動かしながら直感的に理解するためのインタラクティブな学習ページです。

## 開き方

ビルドや依存関係は不要です。`index.html` をブラウザで開くだけで動きます。

```
# 例: ローカルサーバーで開く場合
python3 -m http.server
# → http://localhost:8000/perceptron/ にアクセス
```

## 内容

1. **そもそも何？** — 脳のニューロンの例えで概要をつかむ
2. **仕組みの図解** — 入力 → 重み付き和 → 活性化 → 出力 の流れ
3. **実験ラボ** — スライダーで重み・バイアスを操作し、決定境界の直線をリアルタイムに動かす。グラフをクリックして点も追加できる（Shift+クリックでもう片方のクラス）
4. **学習デモ** — パーセプトロン学習則を「自動／1ステップ」で実行し、直線が自分で動いて2クラスを分けていく様子を観察
5. **論理ゲート** — AND / OR / XOR を切り替え、XOR がなぜ解けないか（線形分離不可能）を体感
6. **数式でおさらい** — 加重和・ステップ関数・学習則を初心者向けに解説

## 技術メモ

- 単一HTMLファイル（CSS・JSを内包）。外部ライブラリなし。
- 可視化は Canvas 2D。座標は \[-1, 1\] × \[-1, 1\] のデータ空間で扱う。
- 学習則: `w ← w + η(t − y)x`、`b ← b + η(t − y)`（t=正解, y=予測, η=学習率）。
