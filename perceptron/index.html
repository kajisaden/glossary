パーセプトロンを体験して学ぼう

パーセプトロンは、ニューラルネットワーク（AIの脳）のいちばん基本となる部品です。
スライダーを動かしたり、学習を実行したりしながら、手を動かして直感的に理解しましょう。

① そもそも何？ ② 仕組み ③ 実験する ④ 学習させる ⑤ 論理ゲート ⑥ 数式

1パーセプトロンって、そもそも何？

私たちの脳は ニューロン（神経細胞） がつながってできています。1つのニューロンは、 たくさんの信号を受け取り、それを合計して「強ければ発火する／弱ければ何もしない」を決めます。

この仕組みをコンピュータでまねた、いちばんシンプルなモデルが パーセプトロン です。 やっていることは、たったこれだけ：

📥 いくつかの入力を受け取る → ✖️ それぞれに重みをかけて足す → ⚖️ 合計がしきい値を超えたら 「はい(1)」、超えなければ 「いいえ(0)」 と答える。

たとえば「傘を持っていく？」を決めるとき、降水確率 と 空のくもり具合 を見て、 重要な要素を重く見て（重み）、総合点がある基準を超えたら「持っていく」と判断する——そんなイメージです。

4パーセプトロンに「学習」させてみよう

手で直線を合わせるのは大変ですよね。実は、パーセプトロンは 自分で重みを調整して 正解に近づくことができます。これが「学習」です。

やり方はシンプル：点を1つずつ見て、間違えたらその点を正しく分類する方向へ重みを少しだけ動かす——これを繰り返すだけ。

5論理ゲートで限界を知ろう

パーセプトロンは AND（かつ）や OR（または）のような論理演算を学習できます。 でも XOR（排他的論理和） だけはどうしても解けません。なぜでしょう？

x₁x₂正解

AND と OR は、●と●を 1本の直線で分けられます（＝線形分離可能）。 一方 XOR は、対角どうしが同じクラスなので、どんな直線を引いても分けられません。 これが「1層のパーセプトロンの限界」で、層を重ねた 多層パーセプトロン が生まれたきっかけです。

6数式でおさらい

ここまでの内容を、改めて式で確認します。難しく見えますが、やっていることは「重み付きの足し算」と「比較」だけです。

u = w₁x₁ + w₂x₂ + b

u は「重み付き和（加重和）」。各入力に重みをかけて、バイアスを足したものです。

y = step(u) =   1  (u ≥ 0 のとき) ,   0  (u < 0 のとき)

step は活性化関数。ある基準（0）を境に、出力を 0 か 1 にパキッと切り替えます。

そして学習則（重みの更新）はこうです。正解を t、予測を y、学習率を η とすると：

wᵢ ← wᵢ + η (t − y) xᵢ  ,   b ← b + η (t − y)

予測が当たっていれば (t − y) = 0 なので何も変わりません。外れたときだけ、 正解に近づく方向へ重みを少し動かす——セクション④でやっていたのは、まさにこの計算です。

🎓 まとめ：パーセプトロンは「入力に重みをかけて足し、しきい値で0/1を出す」だけのシンプルな仕組み。 間違えるたびに重みを少しずつ直すことで、データを分ける直線を自分で見つけます。 この小さな部品を何層も重ねたものが、今のディープラーニングの土台になっています。
